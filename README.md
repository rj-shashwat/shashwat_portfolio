## 🚀 Featured Backend Architectures

Here are some of the core distributed backend projects showcased in this portfolio:

1. **Distributed E-Commerce Microservices Platform**
   * *Architecture*: 8 decoupled Spring Boot services orchestrated via Spring Cloud Gateway and Eureka Discovery.
   * *Highlights*: Asynchronous order fulfillment using Apache Kafka, Redis distributed caching for 40% query latency reduction, and stateless JWT authentication.
2. **High-Throughput Financial Transaction Engine**
   * *Architecture*: Event-sourced payment ledger engineered for ACID compliance and idempotent API endpoints.
   * *Highlights*: Handles concurrent transactions with optimistic locking, real-time fraud detection pipelines, and Prometheus/Grafana observability.
3. **Lovable Clone // AI Web Generator Backend**
   * *Architecture*: Scalable code-generation API leveraging Spring Boot webhooks and asynchronous worker queues.
   * *Highlights*: Real-time Server-Sent Events (SSE) streaming for code synthesis and Dockerized sandbox execution environments.

## 💻 Local Development Setup

To run or modify this project locally on your machine:

```bash
# 1. Clone the repository
git clone https://github.com/rj-shashwat/shashwat_portfolio.git

# 2. Navigate into the directory
cd shashwat_portfolio

# 3. Install dependencies
npm install

# 4. Start the local development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your web browser to view the portfolio.
